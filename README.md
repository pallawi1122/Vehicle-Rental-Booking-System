# 🚗 Vehicle Rental Booking System

A Full Stack Vehicle Rental Booking System developed using **Python Flask, SQLite, HTML, CSS, and JavaScript**. This project allows users to register, log in, book vehicles, make payments, and manage bookings through a simple and user-friendly interface.

## 📌 Features

- 👤 User Registration & Login
- 🔐 Session Management
- 🚗 Vehicle Booking (Car, Bike, Scooter)
- 💳 Payment System (UPI, Debit Card, Credit Card, Cash)
- 🧾 Booking Receipt
- 📋 Booking History
- 🔍 Search Bookings
- ✏️ Update Booking
- ❌ Delete Booking
- 👨‍💼 Admin Dashboard
- 📊 Total Users, Total Bookings & Revenue
- 👤 User Profile
- 📱 Responsive Design

## 🛠️ Technologies Used

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/pallawi1122/Vehicle-Rental-Booking-System.git
```

2. Open the project folder

```bash
cd Vehicle-Rental-Booking-System
```

3. Install Flask

```bash
pip install flask
```

4. Run the project

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
